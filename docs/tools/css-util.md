### 描述
常用样式工具

### 说明
对于方向的缩写

| 方向 | 缩写 |
| :---- | :---- |
| left | l |
| right | r |
| top | t |
| bottom | b |
| left,right | h |
| top, bottom | v |
| top,left,rigth,bottom | f |


### margin, padding，font-size
范围：0-64
步长：2

```html
<!-- margin-top: 10px -->
<div class="mg-t-10"></div>
<!-- pading-left: 20px; padding-right: 20px -->
<div class="pd-h-20"></div>
<!-- margin: 30px -->
<div class="mg-f-30"></div>
<!-- font-size: 12px -->
<div class="fs-14"></div>
```
